Documentation

Introduction

Rocket-UI is a modern CSS library for building responsive, accessible, and beautiful web interfaces. It provides a rich set of components and utilities to speed up your development workflow.

How to Use (CDN)

CDN Import

Include the following CDN links in your HTML <head> to use Rocket-UI instantly:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Anmol-rocket/Rocket-UI@main/src/_base.css">
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Anmol-rocket/Rocket-UI@main/src/index.css">
<script src="https://cdn.jsdelivr.net/gh/Anmol-rocket/Rocket-UI@main/src/rocket-ui-sw.js"></script>

Automatic Caching (Offline Support)

To enable offline caching of Rocket-UI CSS files, use the provided Service Worker. It will cache the CDN files and serve them when offline:

// rocket-ui-sw.js
const CACHE_NAME = 'rocket-ui-cache-v1';
const FILES_TO_CACHE = [
  'https://cdn.jsdelivr.net/gh/Anmol-rocket/Rocket-UI@main/src/_base.css',
  'https://cdn.jsdelivr.net/gh/Anmol-rocket/Rocket-UI@main/src/index.css'
];
self.addEventListener('install', event => {
  event.waitUntil(
    caches.open(CACHE_NAME).then(cache => cache.addAll(FILES_TO_CACHE))
  );
});
self.addEventListener('fetch', event => {
  event.respondWith(
    caches.match(event.request).then(response => {
      return response || fetch(event.request);
    })
  );
});

Register the Service Worker in your main JS file:

if ('serviceWorker' in navigator) {
  navigator.serviceWorker.register('rocket-ui-sw.js');
}

Usage

Apply the provided classes to your HTML elements. For example, to create a button:

<button class="btn btn-primary">Click Me</button>

Customization

Override CSS variables or add your own styles to customize the look and feel. Refer to the Utilities section for available utility classes.

Example

<div class="card">
  <h3>Card Title</h3>
  <p>Card content goes here.</p>
</div>
